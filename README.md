# txt2srt

Простой инструмент для преобразования текстовых транскрибаций в файлы субтитров SRT. 
Автоматически разбивает текст на сегменты оптимальной длины и распределяет их по времени.

Возможности:
- Автоматическая разбивка длинного текста на читаемые сегменты
- Настраиваемая скорость чтения и длина субтитров
- Поддержка различных форматов ввода (txt, json)
- CLI и Python API
